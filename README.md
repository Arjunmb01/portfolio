Welcome to my personal **Full Stack Developer Portfolio**.

## 🚀 Project Overview

This project is a modern, responsive, and animated portfolio website built with **React**, **TypeScript**, **Tailwind CSS**, and **shadcn-ui**.  
It highlights my journey as a developer, my tech stack, and some of the projects I’ve developed.

---

## ✨ Features

- 🏠 **Hero Section** – My name, role, and a professional tagline.  
- 👨‍💻 **About Section** – A brief intro about who I am and what I do.  
- ⚙️ **Skills Section** – Showcases technologies like:
  - Frontend: HTML, CSS, JavaScript, React, TailwindCSS  
  - Backend: Node.js, Express.js  
  - Database: MongoDB  
  - Tools: Git, GitHub, Docker, AWS (basic)  
- 💼 **Projects Section** – Highlights key projects:
  - **TaskPulse** – Smart TODO app with reminders (React, Node.js, MongoDB, Bree.js)  
  - **LaptopHub** – Laptop e-commerce platform with Razorpay integration (Node.js, Express, MongoDB, EJS)  
  - **WeatherApp** – Real-time weather app using OpenWeather API  
- 📚 **Experience Section** – Learning journey from Brototype bootcamp and personal projects.  
- 📬 **Contact Section** – Simple contact form and social media links.  
- 🌗 **Dark/Light Mode** – Toggle for modern theme experience.

## ⚙️ Getting Started

Follow these steps to set up and run the project locally:

```bash
# Step 1: Clone the repository
git clone <YOUR_REPO_URL>

# Step 2: Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev
